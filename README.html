gfuzzy is a simple Groovy library that provides an implementation of Fuzzy Logic.

The project currently has 2 sub-projects, gfuzzy-core, which implements the library itself, and gfuzzy-example, which shows how it can be used.

The library can be built and its tests run through Gradle. See build.gradle in the main project for details.

Eclipse support is provided for use with Groovy-Eclipse. Eclipse users should set a path variable, GRADLE_M5_CACHE, to point to the cached artifacts in their %USER_HOME% directory, i.e. %USER_HOME%/.gradle/caches/artifacts-3.
